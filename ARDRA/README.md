# ARDRA — Drones Decoded 🚁

Website for **Team ARDRA's** ideathon: **Drones Decoded**

## How to Run

Just open `index.html` in a browser — no build steps or dependencies needed.

Or use a local server:
```bash
npx serve
```

## Files

| File | Description |
|------|-------------|
| `index.html` | Page structure and content |
| `style.css` | Styling and animations |
| `script.js` | Interactivity (particles, accordions, scroll effects) |

## Tech Stack

Pure HTML + CSS + JavaScript — no frameworks.
